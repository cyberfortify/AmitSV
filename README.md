# Amit Subhash Vishwakarma - Portfolio Website

A modern, responsive portfolio website for a Senior PMO Engineer with light/dark mode toggle.

## 🌟 Features

- **Responsive Design** - Works perfectly on mobile, tablet, and desktop
- **Light/Dark Mode** - Toggle between themes with persistent preference
- **Graffiti-style Background** - Unique visual design
- **Smooth Animations** - Scroll-triggered animations
- **Contact Form** - Opens email client with pre-filled message
- **Modern UI** - Clean, professional look with accent color #CCFF33

## 📄 Sections

- **Hero** - Name, title, and CTA buttons
- **About** - Strategic planning, Operational excellence, Stakeholder management
- **Experience** - Timeline of work history (KPIT, EV Techo, Foton Motors, GM)
- **Skills** - Project Management, Operations, Tools & Systems
- **Education** - Academic qualifications
- **Awards & Languages** - Achievements and language proficiency
- **Projects** - Key achievements and accomplishments
- **Contact** - Email, phone, LinkedIn, and contact form

## 🛠️ Tech Stack

- HTML5
- CSS3 (with CSS variables for theming)
- JavaScript (vanilla)
- Google Fonts (Outfit, DM Sans)

## 🚀 Getting Started

1. Clone the repository:
```bash
git clone https://github.com/cyberfortify/AmitSV.git
```

2. Open `index.html` in your browser

## 📱 Responsive Breakpoints

- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

## 🎨 Color Theme

- **Primary Accent:** #CCFF33 (Lime Green)
- **Light Mode Background:** #f5f9e6
- **Dark Mode Background:** #0a0a0a
